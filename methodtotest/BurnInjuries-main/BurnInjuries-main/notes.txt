Now we just look at the burn data:
The following characteristics are interesting for us:
- time point (hr0,... hr168, plus D14 and D21)
- covariates (age, gender)
- TBSA

PIGLASSO (graphical LASSO) infers an undirected conditional-dependence network from a set of samples. 
It does not directly model temporal causality unless you encode time somehow (e.g., separate networks per timepoint, windows, or “delta” features). 
So in practice you choose how to slice samples into groups, then infer a network per group, then compare them via diffusion / knockout.

Idea is to do time-binning and infer networks per bin, stratified by outcome.

We first bin by age:
      18–40  -> YngAdult
      41–64  -> MidAdult
      65–85  -> Elder
Then we bin by TBSA:
      Mild:     <10
      Moderate: 10–20
      Severe:   20–40
      Massive:  >40

Then we bin by timepoints into:
- T0: hr0 (baseline)
- Early: hr1–h24
- Mid: hr24–hr96
- Late: hr96–hr168
- Follow-up: >hr168

We then combine timepoints into 3 different phases:
- Acute: T0 + Early + Mid
- Proliferation: Late 
- Remodelling: Follow-up

filter_burn.py

Now we have to do gene filtering per Acute, Proliferation and Remodelling phase for each age&TBSA group. We do:
- cross-dataset intersection between burn and trauma data
- mean expression filtering
- mean-variance filtering (keep genes with variance in the top 25 percentile) > This is an optional step! Turned off by default.
